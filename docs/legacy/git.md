## Git training

Franklin once created a very basic Git tutorial targeted toward
immediate basic needs, but there are better in-depth learning
materials available. For example, there are free online courses such
as:

- The free [GitHub Learning Lab](https://lab.github.com/)
- A free Coursera course [Introduction to Git and
GitHub](https://www.coursera.org/learn/introduction-git-github).

The hardest thing about Git is its terrible names for commands and
other terminology. Otherwise, it is conceptually a very elegant and
powerful tool.
