# TypeScript

[TypeScript](https://www.typescriptlang.org/) is "JavaScript that
scales": a superset of the language that provides static typing while
generating ordinary JavaScript for compatibility.

## Use

Franklin uses TypeScript for the [TalkBank API](talkbank-api.md) that
John uses for TalkBank DB generation.

## Installation on [macOS](macos.md)

Project-localized installation of TypeScript uses `package.json`.

One way to install TypeScript globally is to use [Homebrew](homebrew.md).

``` shellsession
$ brew install typescript
```
