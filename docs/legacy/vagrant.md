# Vagrant

HashiCorp [Vagrant](https://www.vagrantup.com/) is "a tool for
building and managing virtual machine environments in a single
workflow".

We currently use it in conjunction with [VirtualBox](virtualbox.md).

## Installation on [macOS](macos.md)

``` shellsession
$ brew cask install vagrant
```
