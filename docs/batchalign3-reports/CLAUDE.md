# CLAUDE.md — batchalign3 Private Reports

This tree is private archival material moved out of the public `batchalign3`
repo.

## Rules

- Do not move material back into the public repo unless it is rewritten as
  current public documentation or a public migration note.
- Prefer concise index pages in the public repo over restoring large internal
  reports.
- Keep filenames descriptive and date-stamped when the document is a one-off
  audit or execution report.
- Preserve original context when moving material here; if a public page was
  trimmed, note the original public source near the top of the archived file.

## Organization

- `decisions/` is for proposals, spikes, and internal architectural debate.
- `reference/` is for bug forensics, benchmark reports, and deep implementation
  analyses.
- `developer/` is for internal workflow, tooling, and documentation governance.
- `migration/` is only for private migration archaeology; public migration docs
  belong in `batchalign3/book/src/migration/`.
