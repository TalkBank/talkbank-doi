# batchalign3 Private Reports

Archived batchalign3 material that should not remain in the public repo lives
here.

## Purpose

Use this tree for:
- superseded proposals and planning notes
- internal audits and comparison reports
- historical implementation notes that are not needed by public users
- migration-adjacent archaeology that is too detailed for the public book

## Layout

- `architecture/` — retired architecture notes and extracted historical sections
- `decisions/` — proposals, spikes, and internal decision records
- `developer/` — private workflow notes and documentation audits
- `migration/` — migration-adjacent internal history not needed publicly
- `reference/` — analysis reports, bug forensics, and historical reference notes
- `root/` — moved top-level repo notes

## Public boundary

Anything needed by a new `batchalign3` user or a `batchalign2` migrator should
stay in the public `batchalign3/book/src/` tree. Material here is private
archive, not public release documentation.
