## Scope and standard

The goal is not just public-doc cleanup. The current standard is:

- public docs must be technically accurate and current
- private docs must also be technically accurate unless explicitly marked
  historical or no longer relevant
- valuable private material should be promoted into public docs after rewrite
  and verification
- irrelevant/private/historical provenance should be extracted surgically, not
  discarded
- migration guidance must distinguish:
  - same workflow
  - compatible replacement
  - deliberate divergence

Authoritative external references called out during the audit:

- CLAN manual: <https://talkbank.org/0info/manuals/CLAN.html>
- CHAT manual: <https://talkbank.org/0info/manuals/CHAT.html>
