## Services

### nginx (web server)

Primary web server. Serves TalkBank web content and proxies CGI requests.

```bash
systemctl status nginx
sudo nginx -t -c /etc/nginx/nginx.conf   # test config
sudo systemctl reload nginx               # reload after config changes
```

Config: `/etc/nginx/conf.d/talkbank.conf` (generated by `webdev/`, see below).

### fcgiwrap (CGI execution)

Wraps legacy Perl CGI scripts as FastCGI, invoked by nginx. **This is what runs
`morgra2jpg.cgi` — NOT Apache.**

```bash
systemctl status fcgiwrap
ls -la /var/run/fcgiwrap.socket          # must exist, owned by www-data
```

If fcgiwrap is missing after a package change:
```bash
sudo apt install fcgiwrap
sudo systemctl enable --now fcgiwrap
```

### morgra2jpg.cgi (MOR/GRA diagram renderer)

Perl CGI script that renders `%mor` + `%gra` dependency trees as JPEG images using
Graphviz. Used by TalkBank web pages (e.g., Miyata corpus, childes-bank).

**Location:** `/usr/lib/cgi-bin/morgra2jpg.cgi`
**Source repo:** `TalkBank/gra-cgi` (GitHub), cloned at `~/talkbank/gra-cgi/`

**Dependencies:** `graphviz`, `perl`, `fcgiwrap`

**nginx routing** (from `webdev/conf.d/talkbank.conf`):
```nginx
location ~ ^/cgi-bin/(.*)$ {
    gzip off;
    fastcgi_pass unix:/var/run/fcgiwrap.socket;
    include fastcgi_params;
    fastcgi_param SCRIPT_FILENAME /usr/lib/cgi-bin/$1;
    fastcgi_param SCRIPT_NAME /cgi-bin/$1;
    fastcgi_param DOCUMENT_ROOT /usr/lib/cgi-bin;
}
```

**To update the CGI script:**
```bash
cd ~/talkbank/gra-cgi && git pull
sudo cp -p morgra2jpg.cgi /usr/lib/cgi-bin/
```

**To verify it works:**
```bash
curl -s "https://talkbank.org/cgi-bin/morgra2jpg.cgi" -o /dev/null -w "%{http_code}"
# Should return 200 or 400 (missing params), NOT 502/503
```

### Apache — NOT USED

Apache was purged from this server on 2026-03-12 during the Ubuntu 26.04 upgrade.
All CGI execution goes through nginx + fcgiwrap. **Do not reinstall Apache.**

The legacy `homebank.md` doc references Apache for the *homebank* server (different
machine). That does not apply to net/talkbank.

### Tailscale (mesh VPN)

Provides remote access to net via the `talkbank` hostname.

**Apt repository:** Must match the Ubuntu codename. After OS upgrades, the apt source
needs updating.

```bash
# Check current sources:
ls /etc/apt/sources.list.d/tailscale*

# Fix after an OS upgrade (replace CODENAME with actual, e.g., "resolute"):
curl -fsSL https://pkgs.tailscale.com/stable/ubuntu/CODENAME.noarmor.gpg \
  | sudo tee /usr/share/keyrings/tailscale-archive-keyring.gpg >/dev/null
echo "deb [signed-by=/usr/share/keyrings/tailscale-archive-keyring.gpg] \
  https://pkgs.tailscale.com/stable/ubuntu CODENAME main" \
  | sudo tee /etc/apt/sources.list.d/tailscale.list

# Clean up upgrade leftovers:
sudo rm -f /etc/apt/sources.list.d/tailscale.list.{migrate,disabled,distUpgrade}
```

**Supported Ubuntu codenames:** Check https://pkgs.tailscale.com/stable/ — as of
2026-03-12, all codenames through `resolute` (26.04) are available.

## OS Upgrade Checklist

When upgrading Ubuntu on this machine:

1. **Before upgrading:**
   - Stop batchalign services
   - Note which services are running: `systemctl list-units --type=service --state=running`
   - Back up nginx config: `sudo cp /etc/nginx/conf.d/talkbank.conf ~/talkbank.conf.bak`

2. **During upgrade:**
   - If prompted about config files, keep the local version (our customizations)
   - Apache may be flagged for removal — **this is fine**, we don't use it
   - If interrupted, fix with: `sudo dpkg --configure -a && sudo apt --fix-broken install`

3. **After upgrading:**
   - Verify nginx: `sudo nginx -t && sudo systemctl status nginx`
   - Verify fcgiwrap: `systemctl status fcgiwrap && ls -la /var/run/fcgiwrap.socket`
   - Verify CGI scripts: `ls -la /usr/lib/cgi-bin/morgra2jpg*.cgi`
   - Fix Tailscale apt source (see above) — codename changes every release
   - Clean up `.migrate`, `.disabled`, `.distUpgrade` files in `/etc/apt/sources.list.d/`
   - Run `sudo apt update && sudo apt upgrade` to catch stragglers
   - Restart batchalign services
