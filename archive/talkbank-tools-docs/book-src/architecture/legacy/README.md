# rust - Workspace Overview

## Overview
Rust workspace for CHAT parsing, validation, CLI tooling, and LSP support.

## Key Commands
```bash
cargo check --all-targets
cargo test
cargo nextest run -p talkbank-parser-tests
```

## Status and Limitations
- Tree-sitter parser is the reference implementation.
- Direct parser parity and performance work is still in progress.
